# hopsell-website 跳躍拍賣

 * 跳躍拍賣的目的在於，將自己使用不到的物品轉售給其他更需要的人。
 * 衣服為例：在這個快時尚的時代，每週都有新品不斷上市，讓已經穿不到的衣服，能上架給其他人選購。
 * 不僅達到資源有效循環，也減少不必要的浪費。

## 主要功能：

| 商品系統 | 商品瀏覽 | 商品分類 | 喜好商品收藏 | 商品上傳 | 關鍵字搜尋 |
| -------- | -------- | -------- | -------- | -------- |-------- |
| 會員系統  | 個人賣場管理 | 資料設定修改 | 購物車  | 訂單管理  |   |
